# Simulazione d'esame del 5 dicembre 2025 
Simulazione d'esame del 5 dicembre 2025 presso l'ITS Angelo Rizzoli corso 3D Simulation and Metaverse specialist.
<p>Si tratta di un progetto Unity, realizzato con il template MR, in cui l'utente pu√≤ spegnere o accendere una lampada da notte posta su un comodino, in un'ambientazione notturna, mediante pannello UI.</p>

# Come installare e scaricare questo repo
<ol>
  <li>
    Scaricare la versione Unity 6000.2.6f2.
  </li>
  <li>
    Clonare il repo all'interno di una cartella.
  </li>
  <li>
    Aprire il progetto da UnityHub assicurandosi di avere la corretta versione dell'editor installata e selezionata.
  </li>
</ol>
